# Liquid Diagrams

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'liquid-diagrams'
```

And then execute:

```bash
$ bundle install
```

Or install it yourself as:

```bash
$ gem install liquid-diagrams
```

## Usage

```ruby
require 'liquid-diagrams'

# or

require 'liquid_diagrams'
```
